SELTRA

Specifics (above)

Version 7

Stainless Steel, 5.122cm x 1.254cm x 2.5cm

Silicon strap, 18mm joint

Filled with epoxy, gold painting & sculpture

Seltra is a name given to each of my attempts to create the "perfectly beautiful object." There were a total of four attempts, starting from 2008. After my last attempt, I concluded that it is impossible to create a perfectly beautiful object. Afterwards, I embedded a tattoo on my arm that symbolizes all the attempts.

Significance

Although it was a failure, this project established two core values in my life. The first is, "Do not give up." This project lasted eight years; by the time I was finished with the last version, this project had took up almost half of my lifetime. Through all those years, I studied new skills to improve my product: woodworking, jeweling, CAD, and many other skills I wouldn't have learned if not for this project. I realized that anything can be achieved as long as you put your mind to it, no matter how hard the task seems at the time. The second is, "Everything will change." This project reached a halt when I realized that my definition of beauty will change through time. You cannot create a "perfectly beautiful object" without having a permanent definition of beauty. I decided to cancel this project, accepting that even what I believe to be perfect will eventually be not the case.
